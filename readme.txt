A SIP Calculator with customizable default values for Monthly Investment, Expected Return Rate, and Time Period.

== Description ==

The SIP (Systematic Investment Plan) Calculator plugin allows users to easily calculate returns based on customizable inputs for Monthly Investment, Expected Return Rate, and Investment Time Period. This plugin is simple, lightweight, and customizable for any WordPress site.

== Features ==

* **Customizable Default Values:**
   * Set default values for Monthly Investment, Expected Return Rate, and Time Period from the admin panel.
   
* **User-Friendly Calculator Interface:**
   * Clean and responsive design that works on all devices.
   * Simple input fields for Monthly Investment, Expected Return Rate, and Time Period.

* **SIP Results Display:**
   * Output displays the Total Investment, Estimated Returns, and Maturity Value.
   * Graphical representation.

* **Shortcode Support:**
   * Embed the calculator on any page or post with a shortcode: `[kp_sip_calculator]`.

* **Customization Options for Users:**
   * Users can adjust the Monthly Investment, Return Rate, and Time Period to get updated results instantly.

* **Admin Control Panel:**
   * Easily set default values and manage calculator settings from the WordPress admin dashboard.

* **Responsive and Mobile-Friendly:**
   * Fully optimized for mobile devices and tablets for a seamless user experience.

* **Lightweight and Fast:**
   * Minimal impact on site loading times with fast calculations.

* **Cross-Browser Compatibility:**
   * Tested with major browsers including Chrome, Firefox, Safari, and Edge.

* **Localization and Translation Ready:**
   * Translation-ready, with support for various languages using WordPress's translation features.

* **Security and Performance:**
   * Follows best security practices with proper data sanitization and validation.

== Installation ==

= Installation from within WordPress =

1. Visit **Plugins > Add New**.
2. Search for **KP SIP Calculator**.
3. Install and activate the KP SIP Calculator plugin.
4. Use the `[kp_sip_calculator]` shortcode to display the calculator on any page or post.
5. Go to the 'SIP Calculator' settings page to configure the default values and other settings.

= Manual installation =

1. Upload the entire `kp-sip-calculator` folder to the `/wp-content/plugins/` directory.
2. Visit **Plugins**.
3. Activate the KP SIP Calculator plugin.
4. Use the `[kp_sip_calculator]` shortcode to display the calculator on any page or post.
5. Go to the 'SIP Calculator' settings page to configure the default values and other settings.

== Frequently Asked Questions ==

= How do I change the default values for the calculator? =

You can change the default values for Monthly Investment, Expected Return Rate, and Time Period from the plugin settings in the WordPress admin dashboard.

= Can I add the calculator to any page or post? =

Yes! Use the `[kp_sip_calculator]` shortcode to embed the calculator anywhere on your WordPress site.
